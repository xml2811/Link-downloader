# Link Downloader v1.0.0

Portable Windows link downloader by MPTech Tools.

## Recommended download

Download the portable version:

**MPTech-Link-Downloader-v1.0.0-portable.exe**

The portable version does not require installation.

If included, the installer version is optional:

**MPTech-Link-Downloader-v1.0.0-installer.exe**
